# 📌 Overview

NetflixGPT is an AI-powered movie recommendation platform that combines **Large Language Models (LLMs)** with the **TMDB API** to provide personalized movie recommendations using natural language.

Instead of manually searching movies, users can simply describe what they want to watch and the AI recommends relevant movies intelligently.
